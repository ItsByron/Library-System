# Library Management System

## Requirements
- Node.js
- MySQL

## Setup Instructions

### 1. Clone the repository
git clone https://github.com/yourusername/LibrarySystem.git

### 2. Install packages
npm install

### 3. Setup Database
- Open MySQL Workbench
- Run the database.sql file

### 4. Setup Environment
- Create a .env file
- Copy this inside:
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=LibrarySystem

### 5. Run the system
node server.js

### 6. Open browser
http://localhost:3000
